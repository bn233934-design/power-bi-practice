# power-bi-practice
A beginner-friendly Power BI project showcasing an **interactive sales dashboard**.  
Visualizes sales trends, top products, and regional performance with clear KPIs.

## Features
- Cleaned and transformed data using Power Query
- Calculated KPIs with DAX
- Interactive visuals with slicers and charts
- Sales analysis by product, category, and region

## Tools Used
- Power BI Desktop
- Excel (raw data)
- DAX formulas
